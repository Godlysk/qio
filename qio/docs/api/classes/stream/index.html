qio › Globals › Stream

Class: Stream <A1, E1, R1>

Represents a sequence of values that are emitted over time.

Type parameters

▪ A1

The value that would be emitted by this stream.

▪ E1

Possible errors that could be thrown by the stream.

▪ R1

Environment needed to execute this instance.

Example:

import {Stream} from '@qio/core'

const s = Stream.of(1, 2, 3).reduce(0, (a, b) => a + b)

const runtime = defaultRuntime()
runtime.execute(s.drain, console.log) // 6
